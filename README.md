# ToolBot — Учёт строительных инструментов

Телеграм-бот для бронирования, сдачи и учёта строительных инструментов через QR-коды.

## Возможности

- Аренда инструмента из общего списка
- Получение QR-кода при бронировании
- Сдача инструмента по фото QR-кода или кнопкой
- Просмотр своих инструментов
- Просмотр инструментов по объектам строительства
- Админ-панель: добавление и удаление инструментов
- Главное меню с кнопками (удобно без команд)

## Технологии

- Python 3.11+
- Aiogram 3.x
- SQLite (aiosqlite)
- OpenCV + pyzbar для распознавания QR
- Pillow для генерации QR-кодов
